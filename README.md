# Almoosa AdminPanel

A comprehensive admin panel for efficient management of healthcare system modules. This project is developed using VueJS for the frontend and NestJS for the backend.

## Clone Repo

```pytho
git clone https://github.com/muhammadjawadofficial/almoosa-adminpanel.git
cd almoosa-adminpanel
```

## Verify Node Version

Use the node version manager nvm or any other version manager to install node version

```bash
node -v
```
It should be v12.x.x. If not then install and use it.
```
nvm install 12
nvm use 12
```

## Install Dependencies

Use the node package manager [npm](https://www.npmjs.com/) to install libraries.

```bash
npm install
```

## Start the development server
```
npm run serve
# or
yarn serve
```

## Usage

After starting the development server, you can access the admin panel at http://localhost:8000 by default. Use the credentials provided to you to log in and start managing the healthcare system modules.
